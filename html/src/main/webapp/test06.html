form요소 - input type의 값들

  1. input type="text" 주관식. 한 줄 글상자
  2. (input에 대한 설명문을 뜻하는 label.
    label태그와 input태그를 for와 id값으로 서로 연결시켜 줌)
    size="n"으로 입력창의 길이를 지정.
    maxlength="n"으로 입력값의 최대치를 제한.
  3. input type="password" 주관식. 암호 글상자
  4. textarea ... /textarea 주관식. 큰 글상자
    cols="n"와 rows="n"로 상자의 가로크기와 세로크기를 지정.
  5. input type="radio" 객관식(단일선택형). 라디오버튼
    (같은 질문하의 요소끼리는 name의 값을 서로 일치시킴) 초기값으로 지정될 요소는 checked(checked="checked")를 추가.

    가장 선호하는 색상은?

  6. input type="checkbox" 객관식(다중 선택형). 체크박스
    초기값으로 지정될 요소는 checked(checked="checked")를 추가.

    선호 색상은?

  7. select ... /select 공간절약형 객관식. 펼침목록(드롭다운리스트)
    select안에 option으로 항목들을 추가함. selected속성으로 초기값을 지정하고

    가장 선호하는 색상은?

    가장 선호하는 색상은? (selected="selected"/selected)

    상담가능한 시간은? (다중선택 multiple="multiple"/multiple 추가)

    상담가능한 시간은? (상자형 size="n" 추가)

    오전과 오후 중 각 상담가능한 시간은? (optgroup태그로 서로 관련있는 option들을 묶어서 그룹화하여 나타낼 수 있고,
    해당 그룹의 주제, 제목 등은 label속성으로 추가)
    오전과 오후 중 각 상담가능한 시간은?

  8. input type="file" 파일 전송용 상자

  9. input type="hidden" 보이지않는 정보 전송용

  10. input type="time" 시간(오전/오후 시,분)을 지정

  11. input type="date" 날짜(연,월,일)를 지정
    min 속성 -> 최소값
    max 속성 -> 최대값
    step 속성-> 증감 간격 지정


  12. input type="datetime-local" 날짜와 시간(연,월,일,오전/오후 시,분)를 지정

  13. input type="week" 날짜(년,주)을 지정

  14. input type="month" 날짜(년,월)을 지정

  15. input type="number" 숫자값을 지정
    min 속성 -> 최소값
    max 속성 -> 최대값
    step 속성-> 증감 간격 지정





  16. input type="range" 범위내의 값을 지정
    min 속성 -> 최소값, max 속성 -> 최대값, step 속성-> 증감 간격 지정

  17. input type="tel" 연락처(전화번호)를 지정
    가상키패드에서 숫자 형식으로 바뀔 수 있고 전화 걸기 가능

  18. input type="email"연락처(이메일)를 지정
    가상키패드에서 @나 영문 형식으로 바뀔 수 있고 웹브라우저가 알맞은 email형식인지 체크해줌(@유무 등)

  19. input type="url" 연락처(URL,웹주소)를 지정
    웹브라우저가 알맞은 웹주소형식인지 체크해줌(http://, https:// 유무)

html5에서 추가된 속성들

  1. autofocus(autofocus="autofocus") 자동 포커싱

  2. autocomplete="on/off" 자동완성 기능을 켜거나 끔

  3. required(required="required") 필수입력 요소로 지정

  4. placeholder="***" 입력예시나 설명문 등 표시


  5. readonly(readonly="readonly") 읽기만 가능한 요소로 지정
    입력값이 전송됨
    당신은 입니다.

  6. disabled(disabled="disabled") 사용불가능한 요소로 지정
    입력값이 전송 안 됨
    당신은 입니다.

fieldset과 legend로 폼요소들 그룹화 하기

fieldset : 폼 요소를 그룹으로 묶는 태그
legend : 그룹으로 묶는 구역에 제목을 붙이는 태그
label : 폼 요소에 레이블(텍스트)을 붙이는 태그
필수입력
선택입력

input type="submit/reset/button" 전송/취소/일반 역할의 버튼 생성. value로 버튼의 글자를 지정


button ... /button 전송 역할의 버튼 생성.
취소 or 일반 버튼은 type="reset/button"을 추가.
시작과 종료 사이에 버튼의 글자를 적고 다른 태그를 추가 가능. (버튼안에 이미지를 넣거나 웹아이콘 적용 가능)

input type="image" 전송용 이미지
src속성으로 이미지 지정, width="n"/height="n"으로 가로/세로 크기 지정 가능
