List - 목록 만들기

목록의 종류

  • 순서가 없는 목록 - <ul>
  • 순서가 있는 목록 - <ol>
  • 정의 목록 - <dl>

1. 순서가 없는 목록 - ul(unordered list)

  • 항목하나하나 li 붙혀주기

  • css
  • js
  • web
  • job
  • java
  • DB
  • jsp
  • red = #ff0000 = rgb(255,0,0)
  • green = #00ff00 = rgb(0,255,0)
  • blue = #0000ff = rgb(0,0,255)
