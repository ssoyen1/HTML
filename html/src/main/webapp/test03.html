글자를 바꾸는 명령들


bold체 - bold체 = <b>
italic체 - italic체 = <i>
bold체 - 중요한 내용이라는 의미의 bold체 = <strong>
italic체 - 강조한다는 의미의 italic체 = <em>
small - 글자 크기를 작게 small = <small>
sup - sup 윗첨자 = <sup>
(ex) 10cm3 3000km2
sub - sub 아랫첨자 = <sub>
(ex) H2O, O3, CO2 mark - 형광펜 mark = <mark>

시작과 종료태그가 서로 감싸듯 해야 함
1. b mark = O
2. b mark = X
3. b mark = O
4. b mark = X
