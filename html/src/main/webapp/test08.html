링크로 나타낼 수 있는 종류(Web Browser)

Web Browser가 처리불가한 종류를 링크걸면( *.hwp, *.zip 등)
img.zip 다운로드 됨.

  • URL
  • web page - *.html or *.htm
  • image or movie - *.jpg/*.gif/*.png or *.mp4/*.webm/*.ogg ( 일반이미지타입, 동영상타입도 링크걸면 나옴)
    (ex) photo trees.mp4
  • text file - *.txt
    (ex) w3c.txt
  • 상대경로로 링크 걸기
    html첫수업 표만들기 수업
  • 절대경로로 링크 걸기
    daum pixabay.com pexels.com

링크를 여는 방식 지정 - target="_self/_blank/_top/_parent"

  1. _self - 기본값. 현재 위치에서 링크를 연다
  2. _blank - 새 탭에서 링크를 연다 (ex) naver
  3. _top - (프레임구조)브라우저가 나뉨없이 전체에서 링크를 연다
  4. _parent - 한단계 상위창에 링크를 연다
